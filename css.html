css语法：

CSS 规则由两个主要的部分构成：选择器、声明

选择器:通常是需要改变样式的 HTML 元素

声明:由一个属性和一个值组成

css盒子模型：

Margin(外边距) - 清除边框外的区域，外边距是透明的
"margin:"可设置外边距填充 当前margin为0px

Border(边框) - 围绕在内边距和内容外的边框
"border:"可设置边框类型大小及颜色 当前边框为 1px 实线 黑色

Padding(内边距) - 清除内容周围的区域，内边距是透明的
"padding:"可设置外边距填充 当前paddong为0px

background-color 属性定义了元素的背景颜色
当前背景颜色为粉色

display属性设置一个元素应如何显示
